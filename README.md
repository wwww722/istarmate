# 心情小驿站 — 本地运行说明

这是本地版：不需要注册任何云账号，不需要联网部署。数据存在项目文件夹里的
`data.db` 文件中（SQLite），第一次启动会自动建好表。

## 怎么跑起来（只需要做一次）

1. **装 Node.js**（如果电脑上还没装过）
   去 https://nodejs.org 下载 LTS 版本，跟普通软件一样下一步下一步装完。
   装完后打开"终端"（Mac叫Terminal，Windows叫命令提示符/PowerShell），输入：
   ```
   node -v
   ```
   能看到一个版本号（比如 v20.x.x）就说明装好了。

2. **打开这个项目文件夹**
   把这个zip解压到任意位置，比如桌面上的 `istarmate` 文件夹。

3. **在终端里进入这个文件夹**
   ```
   cd 桌面/istarmate
   ```
   （具体路径根据你解压到哪里调整，Mac/Windows的终端都支持直接把文件夹拖进终端窗口，
   会自动补全路径）

4. **复制环境变量文件**
   把 `.env.local.example` 复制一份，改名成 `.env.local`（同一个文件夹里）。
   内容不用改，里面已经是能直接用的默认值。

5. **安装依赖**（只需要做一次，可能要等1-2分钟）
   ```
   npm install
   ```

6. **启动网站**
   ```
   npm run dev
   ```
   看到类似 `Local: http://localhost:3000` 的提示后，打开浏览器访问：
   ```
   http://localhost:3000
   ```
   网站就跑起来了。

## 以后每次想用，只需要

打开终端，`cd` 到项目文件夹，输入 `npm run dev`，然后浏览器打开 `http://localhost:3000`。
不用重新 `npm install`（除非你换了电脑或删了 `node_modules` 文件夹）。

## 注意事项

- 这个网址只有你自己的电脑能打开，关掉终端窗口网站就停了，别人不能通过链接访问。
  如果以后想让别人也能用，那一步绕不开"部署到公网"，需要用到 Vercel 之类的平台账号。
- 数据存在 `data.db` 文件里。想清空重新测试，直接删掉这个文件，重启 `npm run dev`
  会自动重新建一个空的。
- 想保留数据迁移到新电脑，把整个项目文件夹（包含 `data.db`）一起拷过去就行。

## 流程一览
注册（邮箱+密码）→ 登录 → 首页（青少年心理咨询AI可用 / AI素养课程开发中）
→ 填称呼年龄性别 → 15题心情自评（含自伤信号安全提醒）→ 每日小剧场
（场景 → 选择回应 → 推进 → 结尾，主题根据问卷里偏弱的维度挑选，每天限玩一次）。

AI对话部分没有接，你说了自己加。可以参考的接入点：
`lib/scenarios.js` 里每个场景的文案，以及 `pages/api/scenario.js` 里能拿到
当天 scenario 和用户 questionnaire，直接传给你自己接的AI当背景信息即可。

## 安全机制说明
问卷里有一题专门覆盖"伤害自己"的信号，触发后立刻显示热线（示例号码 400-161-9995，
正式给人用之前换成你们实际能提供的资源）。这只是兜底提示，不是完整的危机干预机制。
